## Prerequisites
- Nodejs
- MongoDB

## Build Setup

``` bash
# install dependencies
npm install
# or
npm run bootstrap
# clean the node_modules & npm cache
npm clean
# serve with hot reload at localhost:3000
npm dev
# serve once
npm start
```

## CI
```bash
# depoly with build
docker-compose up -d --build
# deploy udate
docker-compose up -d
```
