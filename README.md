# A cool fast REST APIs with Node.js, MongoDB, Fastify and Swagger.

> A Node.js project

## Build Setup

``` bash
# install dependencies
npm install
# or
npm run bootstrap
# clean the node_modules & npm cache
npm clean
# serve with hot reload at localhost:8080
npm start
```
## Prerequisites
- Nodejs
- MongoDB
